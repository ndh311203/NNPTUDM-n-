# Thư mục hình ảnh

## File poster hero section

Để hiển thị poster trên trang chủ, vui lòng:

1. **Lưu file ảnh poster** với tên: `poster.jpg` (hoặc `poster.png`)
2. **Đặt file vào thư mục này**: `wwwroot/images/poster.jpg`

### Yêu cầu:
- **Tên file**: `poster.jpg` hoặc `poster.png`
- **Định dạng**: JPG, PNG, WEBP
- **Kích thước khuyến nghị**: 1920x1080px hoặc lớn hơn (tỷ lệ 16:9)
- **Kích thước file**: Tối đa 2MB để tải nhanh

### File hiện tại:
- File sẽ được sử dụng tại: `/images/poster.jpg`
- Được hiển thị tại hero section của trang chủ (Home/Index.cshtml)

---

**Lưu ý**: Nếu file không tồn tại, hero section sẽ hiển thị background mặc định màu hồng.
